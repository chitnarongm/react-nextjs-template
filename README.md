# NexusStore - Modern E-commerce Template

A high-performance, accessible, and localized e-commerce template built with Next.js, React, and daisyUI.

## Features

- **Next.js 16 (Turbopack)**: Lightning-fast development and optimized production builds.
- **Modern UI**: Built with daisyUI and Tailwind CSS for a premium look and feel.
- **i18n Support**: Full localization support using `next-intl` (English & Thai included).
- **Authentication**: Secure authentication flow with `next-auth`.
- **State Management**: Robust state handling with Zustand.
- **Type Safe**: Fully typed with TypeScript.
- **Testing**: Comprehensive test suite with Jest and React Testing Library.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Scripts

- `npm run dev`: Start development server.
- `npm run build`: Create production build.
- `npm run start`: Start production server.
- `npm run lint`: Run ESLint checks.
- `npm run test`: Run tests with Jest.

## Technologies

- **Framework**: Next.js
- **Styling**: Tailwind CSS & daisyUI
- **Icons**: Lucide React
- **Validation**: Zod
- **Auth**: NextAuth.js
- **i18n**: next-intl
